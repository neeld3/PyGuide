Lesson 4: While Loops in Python

What is a while loop?

A while loop repeats a block of code as long as a specified condition remains True. Unlike a for loop, which runs a set number of times, a while loop continues running until some condition stops it. While loops are useful for situations where we don’t know in advance how many repetitions are needed.

Basic while loop example

count = 0

while count < 5:
    print(count)
    count += 1     # increment to avoid an infinite loop

Output:

0
1
2
3
4

Using break to exit a loop

The break statement stops the loop immediately when a condition is met.

num = 1

while num < 10:
    if num == 5:
        break
    print(num)
    num += 1

Using continue to skip an iteration

The continue statement skips the current loop iteration and moves to the next one.

x = 0

while x < 6:
    x += 1
    if x == 3:
        continue
    print(x)

Output:

1
2
4
5
6

Infinite loops

If the loop condition never becomes false, the loop will run forever. This is called an infinite loop. It is usually caused by forgetting to update the condition inside the loop. 

# WARNING: This will run forever
while True:
    print("Stop me with Ctrl + C")

While loop with user input

While loops are commonly used to repeatedly ask for input until the user enters a valid response.

password = ""

while password != "python123":
    password = input("Enter password: ")

print("Access granted!")

Using while loops with lists

numbers = [10, 20, 30, 40]
index = 0

while index < len(numbers):
    print(numbers[index])
    index += 1

Nested while loops

i = 1

while i <= 3:
    j = 1
    while j <= 2:
        print(i, j)
        j += 1
    i += 1

Output:

1 1
1 2
2 1
2 2
3 1
3 2

Why use while loops?

While loops are ideal when you need a loop controlled by a condition rather than a fixed number of iterations. They are useful for validating input, waiting for specific events, processing data streams, and working with situations where you do not know the exact number of repetitions needed in advance.

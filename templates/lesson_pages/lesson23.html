Lesson 9: Functions in Python

What is a function?

A function is a reusable block of code that performs a specific task. Instead of rewriting the same code many times, we define a function once and call it whenever needed. Functions help keep programs organized, efficient, and easier to debug.

How to define a function

def greet():
    print("Hello from a function!")

Calling a function

greet()

Output:

Hello from a function!

Understanding parameters

Parameters allow us to pass information into a function so it can use different values each time it runs. The values we supply when calling a function are called arguments. 

def say_hello(name):
    print("Hello, " + name + "!")

say_hello("Alex")   # Alex is the argument
say_hello("Maya")   # Maya is the argument

Output:

Hello, Alex!
Hello, Maya!

Parameters vs. Arguments

  • Parameters: variables listed in the function definition
  • Arguments: actual values passed when calling the function

Functions with multiple parameters

def multiply(a, b):
    return a * b

print(multiply(4, 6))

Output:

24

Default parameter values

Parameters can have default values, which are used if no argument is provided. 

def welcome(name="Guest"):
    print("Welcome,", name)

welcome()          # uses default value
welcome("Will")    # overrides default

Output:

Welcome, Guest
Welcome, Will

Keyword vs. positional arguments

Arguments can be passed by position or by specifying the parameter name. 

def introduce(name, age):
    print(f"{name} is {age} years old.")

introduce("Sarah", 17)           # positional
introduce(age=18, name="Daniel") # keyword

Returning Multiple Values

def get_stats():
    return 90, 85, 88   # returns a tuple

math, english, science = get_stats()
print(math, english, science)

Variable Scope

Variables created inside a function are local and only accessible within that function. Variables outside a function are global. 

x = 10   # global variable

def show():
    y = 5  # local variable
    print(x, y)

show()
print(x)    # works
print(y)    # error, y does not exist outside the function

Lambda (Anonymous) Functions

A lambda function is a short, one-line function without a name. They are often used when a simple function is needed temporarily. 

square = lambda num: num * num
print(square(6))

Output:

36

Common Mistakes with Functions

  • Forgetting to call the function using ()
  • Returning a value but forgetting to store or print it
  • Not matching the number of parameters and arguments

Why Functions Matter

Functions are essential in real-world programming. They allow us to break programs into logical parts, improve maintainability, avoid repeating code, and build complex systems from smaller building blocks.

Without functions, even small programs quickly become messy and difficult to manage.

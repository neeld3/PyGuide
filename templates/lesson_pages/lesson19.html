Lesson 5: For Loops in Python

What is a for loop?

A for loop lets us repeat a block of code multiple times. It is commonly used to iterate over items in a sequence, such as lists, strings, or ranges of numbers. For loops are essential for processing data, automation, and performing repeated actions efficiently.

Basic for loop example

for i in range(5):
    print(i)

Output:

0
1
2
3
4

Looping through a list

fruits = ["apple", "banana", "cherry"]

for fruit in fruits:
    print(fruit)

Output:

apple
banana
cherry

Looping through a string

word = "Python"

for letter in word:
    print(letter)

Output:

P
y
t
h
o
n

Using range() in loops

range() generates a sequence of numbers. It can take up to three arguments: range(start, stop, step). 

for x in range(1, 10, 2):
    print(x)

Output:

1
3
5
7
9

Using break in a loop

Exits the loop early when a condition is met.

for num in range(10):
    if num == 5:
        break
    print(num)

Using continue in a loop

Skips the current iteration and continues to the next one.

for num in range(6):
    if num == 3:
        continue
    print(num)

Output:

0
1
2
4
5

Nested for loops

We can place a loop inside another loop, useful for working with multi-dimensional data. 

for i in range(3):
    for j in range(2):
        print(i, j)

Output:

0 0
0 1
1 0
1 1
2 0
2 1

Looping with indexes using enumerate()

colors = ["red", "green", "blue"]

for index, color in enumerate(colors):
    print(index, color)

Output:

0 red
1 green
2 blue

Looping through a dictionary

student = {"name": "Alex", "grade": 11, "GPA": 3.8}

for key, value in student.items():
    print(key, ":", value)

Why use for loops?

For loops are essential for automation, searching, sorting, collecting input, analyzing data, and building programs that scale. They allow us to write efficient, clean code instead of repeating lines manually.

Lesson 6: Working with Strings

We can do a lot with strings in Python. So far we have seen that we can print strings, but there is a lot more that is possible.

One thing that we can do is add strings together. For example, maybe I want to add some words together to make a sentence. I can add strings together using the plus sign.

str1 = "I want "
str2 = "pizza."
str3 = "ice cream."
print(str1 + str2)
print(str1 + str3)

When run, it will look like this:

I want pizza.
I want ice cream.

If we want to turn a number into a string in order to add them together, we can turn it into a string using str() and put the variable that isn't a string inside. Here's an example:

myAge = 12
# newString = "I am " + myAge + " years old."
# The line above causes an error! We cannot do string + integer.
correctNewString = "I am " + str(myAge) + " years old."

If we want to see a specific spot in a string, we can use indexing. Indexing allows us to look at a specific character. Slicing is similar, but allows us to look at a group of characters. Here are some examples of both: 

myString = "This is a full sentence!"
letterT = myString[0]
wordThis = myString[:4]
wordFull = myString[10:14]
exclamationMark = myString[-1]

Now that we've seen some examples, lets look at some rules.
1. We can index starting at 0, which is the first character, all the way until the length of the string minus one. This is because we start indexing at 0. if we have the string "hello" and try to index at 5, we will get an error!
2. Slicing is exclusive at the end. That means that the last number we include after the colon will not be added to the string. So if we slice [1:3], we will get index 1 (which is the second character because we start at 0!) and index 2, but not index 3. 3. If we want to start at the beginning of the string, we can leave the space before the colon empty (like this [:5]). If we want to go until the end, we can leave the space after the colon empty (like [10:]). 4. We can use negative indexes! Since 0 represents the first character going from the start, our negative indexes represent going backwards starting with negative one. The second-to-last character is -2, and it just continues from there.

There are very many string methods that we can also use with strings. Here are some examples: 

shortString = "Hi!"
length = len(shortString)
# the len() function lets us find out how many characters are in a string.
# here, it will return 3

myString = "hello hello how are you today?"
numberOfH = myString.count('h')
# count will count how many of that thing exist in the string. 
# for 'h', it will return 3.

firstLetterL = myString.find('l')
# find returns the index of the first time we find that letter. 
# for 'l', it will give us an answer of 2 (remember that indexing starts at 0!).
# if it doesn't find anything that it's looking for, it returns -1!

capitalizedSentence = myString.capitalize()
# capitalize returns the string with the first letter uppercase.
# so, it will return "Hello hello how are you today?"

You can find many more string methods within Python documentation if you are interested, but we will only cover these simpler ones.

Lesson 1: Printing in Python

What does print() do?

The print() function is used to display information on the screen. It is one of the most commonly used functions in Python and is essential for testing, debugging, and interacting with users.

Basic printing

print("Hello, world!")

Output:

Hello, world!

Printing variables

name = "Alex"
age = 17

print(name)
print(age)
print("Name:", name, "- Age:", age)

Output:

Alex
17
Name: Alex - Age: 17

Printing multiple values

We can print multiple items by separating them with commas. Python automatically adds a space between each value. 

print("Apples:", 5, "Oranges:", 3)

Using formatted strings (f-strings)

F-strings offer a clean and modern way to format printed text. They allow variables to be embedded directly inside the string using curly braces {}. 

name = "Maya"
grade = 12

print(f"{name} is in grade {grade}.")

Output:

Maya is in grade 12.

Using escape characters

Escape characters help format text inside strings. 

print("Line 1\nLine 2")   # new line
print("He said, \"Hello!\"")  # quotation marks
print("Path: C:\\Users\\Alex") # backslash

Output:

Line 1
Line 2
He said, "Hello!"
Path: C:\Users\Alex

Changing the ending and separator

# sep changes the separation between printed items
print("A", "B", "C", sep="-")

# end changes what appears at the end instead of a newline
print("Loading", end="...")
print("Done!")

Output:

A-B-C
Loading...Done!

Why printing is important

Printing allows us to understand what our program is doing at each step, display results, debug errors, interact with users, and monitor data. It is one of the most essential tools when learning to program.

Lesson 11: Object-Oriented Programming (OOP)

What is Object-Oriented Programming?

Object-Oriented Programming (OOP) is a programming style that organizes code into objects, which bundle together data and behavior. Instead of writing large blocks of procedural code, OOP models real-world concepts as objects with properties and actions.

For example, if we were modeling a student, we might store information like their name and grade, and give them functions like study() or submit_homework().

The Four Pillars of OOP

  • Encapsulation: bundling data and methods inside a class, controlling access to protect information
  • Abstraction: hiding complex details and exposing only what is necessary to the user
  • Inheritance: allowing one class to receive attributes and methods from another class, enabling code reuse
  • Polymorphism: methods can behave differently depending on the object using them, enabling flexibility and customization

Key OOP Terms

  • Class : a blueprint or template for creating objects
  • Object : an instance of a class
  • Attribute : a variable that stores data within an object
  • Method : a function defined inside a class that describes an object's behavior

Example: Creating a Class and an Object

class Student:
    def __init__(self, name, grade):   # constructor
        self.name = name               # attribute
        self.grade = grade             # attribute

    def display_info(self):
        print(f"Name: {self.name}, Grade: {self.grade}")

student1 = Student("Alex", 11)
student2 = Student("Maya", 12)

student1.display_info()
student2.display_info()

Output:

Name: Alex, Grade: 11
Name: Maya, Grade: 12

Example: Inheritance

class Person:
    def __init__(self, name):
        self.name = name

class Teacher(Person):                  # inherits from Person
    def teach(self):
        print(f"{self.name} is teaching now.")

teacher1 = Teacher("Mr. Johnson")
teacher1.teach()

Encapsulation Example

class BankAccount:
    def __init__(self, balance):
        self.__balance = balance     # private variable

    def deposit(self, amount):
        self.__balance += amount

    def get_balance(self):
        return self.__balance

Why OOP matters in real applications

OOP is widely used in modern software development. It is foundational in game engines, web applications, mobile apps, user interface frameworks, and large-scale engineering projects. Many languages support OOP, including Python, Java, C++, C#, and Swift.

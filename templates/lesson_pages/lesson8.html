Lesson 3: Comparisons and Conditionals

What are comparisons?

Sometimes we want the computer to compare two or more things and tell us whether something is true or false.
A comparison checks how two values relate to each other.
When we compare things, Python gives us either True or False.

Common comparison signs:
== means equal to
!= means not equal to
> means greater than
< means less than
>= means greater than or equal to
<= means less than or equal to

Examples:

print(5 > 3)      # This will show True
print(2 == 4)     # This will show False
print("cat" != "dog")   # This will show True

We can also use additional words along with comparisons. The words not, and, and or are helpful and happen in that order. So the "order of operations" (like PEMDAS in math) is:
Math operations (+, -, *, etc.)
Comparisons (==, <, etc.)
not
and
or

Parenthesis will always come first, just like in regular math expressions!

What are conditionals?

A conditional is a way to tell the computer:
"If something is true, do this. Otherwise, do something else."

In Python we use the word if.
We can also use else for what should happen when the condition is not true.

Example:

age = 10

if age >= 10:
    print("You are at least ten.")
else:
    print("You are younger than ten.")

How conditionals work

Python checks the condition after the word if.

If the condition is True, it runs the code under if.

If the condition is False, it runs the code under else (if there is an else).

More examples:

temperature = 30

if temperature > 25:
    print("It is warm outside.")
else:
    print("It is not very warm.")

name = "Alice"

if name == "Alice":
    print("Hello, Alice!")
else:
    print("Hello, stranger.")

Using if-elif-else

Sometimes we need more than just one if and one else.
We may want the computer to check several different conditions, one after another.
For this, Python uses elif, which means "else if."

Python reads them in order:
It checks the if condition first.

If that is not true, it checks the elif condition.

If none of the conditions are true, it runs the else part (if there is one).

Example:

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: D")

In this program, Python looks at each condition from top to bottom until it finds one that is true.
Once it finds a true condition, it runs that block of code and ignores the rest.

Another example:

color = "green"

if color == "red":
    print("Stop.")
elif color == "yellow":
    print("Slow down.")
elif color == "green":
    print("Go.")
else:
    print("Unknown color.")

Using comparisons and conditionals together

Comparisons help us ask questions.
Conditionals let the computer answer those questions with actions.

Try it yourself!
Think of something you want the computer to check.
For example, you can check if a number is bigger than another number, if two words match, or if someone's score is high enough to win.
Then write an if statement that prints different messages depending on the answer.

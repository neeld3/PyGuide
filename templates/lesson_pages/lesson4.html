Lesson 4: While Loops

What is a while loop?

A while loop is a way to tell the computer: “Keep doing this over and over as long as something is true.” It's like saying, “While it's still daytime, keep playing outside.”

What does a while loop look like?

Here is the correct Python syntax:

while condition:
    # repeat this block

The word while must be lowercase, and the line must end with a colon (:).

A simple example

Let's count from 1 to 5 using a while loop:

counter = 1

while counter <= 5:
    print(counter)
    counter = counter + 1

This loop prints numbers 1 through 5. After each loop, we add 1 to counter so the loop eventually stops.

Why do we need to change a variable?

If we don't change the variable used in the condition, the loop might run forever! This is called an infinite loop, and it happens when the condition never becomes false.

# WARNING: Infinite loop example
number = 1

while number < 5:
    print("This will never stop!")
    # number never changes here!

Because number never increases, it is always less than 5 — so the loop goes on forever.

How to avoid infinite loops

To keep loops safe and working correctly, always make sure the condition can eventually become false. Most loops use something like:

counter = counter + 1

Lesson 6: String Methods & Slicing in Python

What is a string?

A string in Python is a sequence of characters surrounded by quotes. Strings are used to store text, such as names, messages, or any combination of letters, numbers, and symbols. 

text = "Hello, Python!"
print(text)

Accessing characters with indexing

We can access individual characters by using their index position (starting at 0).

word = "Python"
print(word[0])   # first character
print(word[3])   # fourth character

Output:

P
h

String slicing

Slicing allows us to extract parts of a string using the format string[start:end]. The slice includes the start index but stops before the end index. 

phrase = "Programming"
print(phrase[0:6])   # characters 0 to 5
print(phrase[3:])    # from index 3 to end
print(phrase[:5])    # from start to index 4
print(phrase[-3:])   # last 3 characters

Output:

Progra
gramming
Progr
ing

Common string methods

String methods help modify or interact with text easily.

message = "hello world"
print(message.upper())     # HELLO WORLD
print(message.lower())     # hello world
print(message.title())     # Hello World
print(message.capitalize())# Hello world
print(message.replace("world", "Python"))  # hello Python

Removing whitespace

data = "   clean me   "
print(data.strip())   # removes leading & trailing spaces
print(data.lstrip())  # removes left spaces
print(data.rstrip())  # removes right spaces

Splitting and joining strings

text = "apple,banana,cherry"
items = text.split(",")     # split into list
print(items)

joined = "-".join(items)    # join back into string
print(joined)

Output:

['apple', 'banana', 'cherry']
apple-banana-cherry

Checking content

Methods to test what a string contains:

code = "Python3"
print(code.isalpha())   # letters only?
print(code.isdigit())   # digits only?
print(code.isalnum())   # letters or numbers?

Output:

False
False
True

Finding characters or substrings

sentence = "Welcome to Python"
print(sentence.find("Python"))   # index where found
print(sentence.find("Java"))     # -1 means not found

Why strings & methods matter

Strings are used in nearly every program: user input, file names, messages, text processing, formatting, and data cleanup. Understanding slicing and string methods makes working with text efficient and powerful.

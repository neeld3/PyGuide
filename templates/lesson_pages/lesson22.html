Lesson 8: Dictionaries in Python

What is a dictionary?

A dictionary in Python is a collection of key-value pairs. Each value is stored under a unique key, which we use to retrieve information. Dictionaries are useful when we want to store related data together, such as a person's details or settings in a program.

Creating a dictionary

student = {
    "name": "Alex",
    "grade": 11,
    "city": "New York"
}

print(student)

Output:

{'name': 'Alex', 'grade': 11, 'city': 'New York'}

Accessing values in a dictionary

We access values by using their key inside square brackets.

print(student["name"])
print(student["grade"])

Output:

Alex
11

Modifying values

student["grade"] = 12
print(student)

Adding new key-value pairs

student["GPA"] = 3.8
print(student)

Removing items

del student["city"]
print(student)

Useful dictionary methods

print(student.keys())     # list of keys
print(student.values())   # list of values
print(student.items())    # list of key-value pairs

Looping through a dictionary

for key, value in student.items():
    print(key, ":", value)

Output:

name : Alex
grade : 12
GPA : 3.8

Why use dictionaries?

Dictionaries allow us to store and access data efficiently using descriptive keys instead of numeric indexes. They are useful in programs involving user profiles, settings, inventory systems, databases, JSON data, and more.

Lesson 7: Working with Lists

Lists in Python are used to store multiple items in a single variable. You can think of a list like a row of boxes, each holding a piece of information. Lists are very helpful when you want to keep track of many related things at once.

We make a list by putting items inside square brackets [], separated by commas. 

numbers = [1, 2, 3, 4]
fruits = ["apple", "banana", "grape"]
mixedList = ["hello", 3, True]

Just like strings, lists have indexes. Indexing lets us look at a specific item in the list. Remember: indexing starts at 0! 

fruits = ["apple", "banana", "grape"]

firstFruit = fruits[0]    # "apple"
secondFruit = fruits[1]   # "banana"
lastFruit = fruits[-1]    # "grape"

We can also change items in a list. Unlike strings, lists are mutable, meaning we can update them. 

fruits = ["apple", "banana", "grape"]
fruits[1] = "orange"

print(fruits)
# Now the list is: ["apple", "orange", "grape"]

Lists also let us add, remove, and count items. Here are some useful list methods: 

pets = ["dog", "cat", "bird"]

pets.append("fish")     # adds "fish" to the end
pets.remove("cat")      # removes "cat"
numberOfDogs = pets.count("dog")  # counts how many "dog" items are in the list

We can also use len() to find how many items are in a list. 

numbers = [10, 20, 30, 40]
length = len(numbers)   # This will be 4

Slicing works for lists too, just like with strings. Slicing lets us take a part of the list. 

colors = ["red", "blue", "green", "yellow", "purple"]

someColors = colors[1:4]   # ["blue", "green", "yellow"]
firstTwo = colors[:2]      # ["red", "blue"]
lastThree = colors[-3:]    # ["green", "yellow", "purple"]

Lists are one of the most useful tools in Python. They let you store many values, organize them, and work with them easily. As we continue learning, we will use lists very often, so practice reading, changing, slicing, and looping through them!

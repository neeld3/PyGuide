Lesson 3: Conditionals in Python

What are conditionals?

Conditionals allow a program to make decisions based on certain conditions. They let code run differently depending on whether something is true or false. In Python, we use if, elif, and else to control program flow.

Basic if statement

age = 18

if age >= 18:
    print("You are an adult.")

Output:

You are an adult.

if-else statement

temperature = 45

if temperature > 50:
    print("It's warm outside.")
else:
    print("It's cold outside.")

Output:

It's cold outside.

if-elif-else chain

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: D or below")

Output:

Grade: B

Using logical operators

We can combine conditions using and, or, and not.

age = 20
has_id = True

if age >= 18 and has_id:
    print("Entry allowed.")
else:
    print("Entry denied.")

Checking membership with in

colors = ["red", "blue", "green"]

if "blue" in colors:
    print("Blue is available.")

Nested conditionals

x = 10

if x > 0:
    print("Positive number")
    if x % 2 == 0:
        print("Even number")

Output:

Positive number
Even number

Conditional expressions (Ternary operator)

Python also supports a short form of conditional statements on one line.

age = 17
message = "Adult" if age >= 18 else "Minor"
print(message)

Output:

Minor

Why use conditionals?

Conditionals allow programs to react to different situations, validate data, enforce rules, and respond to user choices. They are essential for real-world programs such as games, websites, mobile apps, and automation systems.

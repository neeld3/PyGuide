Lesson 2: Variables in Python

What is a variable?

A variable is a container that stores a value in memory. It allows us to save data and use it later in our program. You can think of a variable as a labeled box where we can put information. 

name = "Alex"
age = 17
score = 95.5

print(name, age, score)

Output:

Alex 17 95.5

Variable naming rules

  • Names must start with a letter or underscore
  • They cannot start with a number
  • They cannot contain spaces or special characters like @, $, %
  • They are case-sensitive (Age and age are different)
  • Use descriptive names to keep programs readable
userName = "John"
username = "Jane"
# These are NOT the same

Changing variable values

Variables can be updated by reassigning them to new values.

count = 5
count = count + 1
print(count)

Output:

6

Different data types in variables

Variables can store many types of data.

text = "Hello"       # string
number = 42          # integer
decimal = 3.14       # float
status = True        # boolean

Checking variable type

value = 10
print(type(value))    # shows the data type

Output:

int

Multiple variable assignment

x, y, z = 1, 2, 3
print(x, y, z)

Output:

1 2 3

Swapping values

a = 10
b = 20

a, b = b, a
print(a, b)

Output:

20 10

Constants

Python does not enforce constants, but by convention we use uppercase names to indicate values that should not change. 

PI = 3.14159
MAX_SPEED = 120

Why variables matter

Without variables, programs cannot store user input, keep track of progress, perform calculations, or remember information. Variables are the foundation of every program, making them essential to learn before moving into more complex topics.

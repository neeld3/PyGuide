Lesson 8: Dictionaries in Python

Dictionaries in Python let us store information in pairs. Each pair has a key and a value. You can think of a dictionary like a real-life dictionary: you look up a word (the key), and it gives you the meaning (the value).

We create a dictionary using curly braces {} with key-value pairs separated by colons. 

student = {
    "name": "Alice",
    "age": 11,
    "favorite_color": "green"
}

To access something inside a dictionary, we use its key. Keys work like labels that point to values. 

student = {"name": "Alice", "age": 11}

print(student["name"])   # "Alice"
print(student["age"])    # 11

Unlike lists, dictionaries do not use numbers for indexing. Instead, they use whatever keys we choose: strings, numbers, or even booleans.

We can add or change items in a dictionary by assigning a value to a key. 

student = {"name": "Alice", "age": 11}

student["age"] = 12             # changes the age
student["favorite_food"] = "pizza"   # adds a new key-value pair

print(student)

We can remove items too, if we no longer need them. 

student = {"name": "Alice", "age": 12, "favorite_food": "pizza"}

del student["favorite_food"]   # removes the key "favorite_food"

Dictionaries also have useful methods that help us explore their data. 

pet = {"type": "dog", "name": "Buddy"}

keys = pet.keys()       # returns all keys
values = pet.values()   # returns all values
items = pet.items()     # returns key-value pairs

Sometimes we want to check whether a key exists in the dictionary before using it. We can do this with the in keyword. 

student = {"name": "Alice", "age": 12}

if "name" in student:
    print("Name is in the dictionary!")

Dictionaries are great when you want to store information about something using labels. For example: 

car = {
    "brand": "Toyota",
    "model": "Camry",
    "year": 2020
}

Now you know how to create, access, change, and explore dictionaries! They are very powerful and helpful for organizing information. You will see dictionaries used everywhere in Python programs.

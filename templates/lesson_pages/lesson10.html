Lesson 5: For Loops

What is a for loop?

A for loop lets the computer repeat something a certain number of times, or go through each item in a collection like a list or a string.

It's like saying: “For each item, do this.”
For loops are very similar to while loops, except we normally use a for loop when we know how many times something should happen for sure. A while loop is normally used when we aren't sure or if we are waiting for something to happen.

What can a for loop go through?

A for loop can iterate through any iterable collection. That means things like:

  • lists
  • strings
  • ranges (made with range())
For example, if you loop through a string, you get each character. If you loop through a list, you get each item.

What does a for loop look like?

The correct Python syntax is:

for item in collection:
    # repeat this block

Most loops use range() to repeat a certain number of times:

for i in range(5):
    print(i)

This will print the numbers 0, 1, 2, 3, and 4.

Examples of for loops

Looping through a list:

fruits = ["apple", "banana", "grape"]

for fruit in fruits:
    print(fruit)

Looping through a string:

for letter in "cat":
    print(letter)

Be very careful not to assume that looping over a string gives you words. Even if you write for word in someString: you will get characters, not words!

Looping using range:

for i in range(3):
    print(i)

This loop prints:

0
1
2

We can even include conditionals like if statements inside for loops, like this:

for i in range(5):
    if i > 1 and i < 4:
        print(i)

Which will print this:

2
3

Why use for loops?

For loops are great when you know how many times you want to repeat something, or when you want to go through every element in a list or string.
They help programs run faster and keep your code neat and organized.

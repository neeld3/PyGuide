Lesson 1: Printing in Python

What does “print” mean in Python?

In Python, print means “show something on the screen.” It allows the computer to show us something using text.

print("Hello!")

When run, it will show:

Hello!

Why do we use printing?

Printing helps us check our work, force the computer to show us something, or see what our program is doing.
We can even print multiple things by putting commas between them. This will be more useful in the future when we start using different types of things in Python.
A comma is automatically replaced by a space, unless we tell Python to use something else. 

print("Hi", "Python programmer!")

The output from above looks like this:

Hi Python programmer!

You can see that the comma created a space, even though there was no space inside our quotation marks.

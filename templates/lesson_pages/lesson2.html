Lesson 2: Variables in Python

What is a variable?

A variable is like a little box where the computer can store information for us. We can put something inside the box, give the box a name, and then use that name later in our program.

Why do we use variables?

Variables help us remember information while our program is running. For example, a variable can remember a number, a word, or even whether something is true or false. They make our code easier to read and easier to change.

Different kinds of variables

1. String Variables

A string is text. It can be a word, a sentence, or even a single letter. We put string text inside quotation marks.

name = "Alice"
print(name)

2. Integer Variables

An integer is a whole number. It has no decimal point. We use integers for things like counting or keeping score.

score = 10
print(score)

3. Float Variables

A float is a number with a decimal point. Floats are useful for measurements, prices, or anything that is not a whole number.

temperature = 23.5
print(temperature)

4. Boolean Variables

A boolean stores either True or False. These are helpful when our program needs to make decisions.

is_raining = False
print(is_raining)

Putting it all together

Here is a short program that uses different kinds of variables at the same time:

name = "Sam"
age = 8
height = 1.3
likes_coding = True

print(name)
print(age)
print(height)
print(likes_coding)

Try it yourself!

Make your own variables by choosing a name and giving it a value. You can make variables for things like your favorite food, your age, the number of pets you have, or whether you like ice cream. Then print them to see what Python remembers for you.

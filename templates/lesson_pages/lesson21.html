Lesson 7: Lists in Python

What is a list?

A list in Python is an ordered collection of items. Lists can store multiple values in a single variable, and the items can be of any data type—numbers, strings, booleans, or even other lists. Lists are extremely useful for organizing and working with groups of related data.

Creating a list

fruits = ["apple", "banana", "cherry"]
print(fruits)

Output:

['apple', 'banana', 'cherry']

Accessing list items

List items are accessed by index, starting at position 0. 

print(fruits[0])   # first item
print(fruits[2])   # third item

Output:

apple
cherry

Modifying list items

fruits[1] = "blueberry"
print(fruits)

Adding items to a list

fruits.append("orange")     # adds to the end
fruits.insert(1, "mango")   # adds at a specific index
print(fruits)

Removing items from a list

fruits.remove("apple")   # remove by value
fruits.pop(2)            # remove by index
print(fruits)

List length

print(len(fruits))

Looping through a list

for fruit in fruits:
    print(fruit)

Sorting and reversing lists

numbers = [5, 1, 8, 3, 2]
numbers.sort()      # ascending
print(numbers)

numbers.reverse()   # reverse order
print(numbers)

Slicing lists

We can access portions of a list using slicing.

letters = ["a", "b", "c", "d", "e"]
print(letters[1:4])   # items at index 1, 2, 3

Output:

['b', 'c', 'd']

Checking if an item exists

if "banana" in fruits:
    print("Banana is in the list!")

Lists with multiple data types

mixed = ["text", 10, True, 4.5]
print(mixed)

Nested Lists (Lists inside lists)

A list can contain other lists, which is useful for storing structured or grid-like data.

matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

print(matrix[0][1])   # row 0, column 1

Output:

2

List Comprehension

List comprehension provides a shorter, cleaner way to create lists using loops and expressions in a single line. 

# regular loop
squares = []
for x in range(5):
    squares.append(x * x)

print(squares)

Using list comprehension:

squares = [x * x for x in range(5)]
print(squares)

Output:

[0, 1, 4, 9, 16]

List Comprehension with Condition

even_numbers = [num for num in range(10) if num % 2 == 0]
print(even_numbers)

Output:

[0, 2, 4, 6, 8]

Why use lists?

Lists are one of the most powerful and frequently used data structures in Python. They allow fast modification, easy grouping of data, and efficient processing with loops and list comprehensions.

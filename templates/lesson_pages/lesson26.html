Lesson 12: Recursion in Python

What is recursion?

Recursion is a programming technique where a function calls itself to solve a problem. The idea is to break a large problem into smaller, simpler versions of the same problem, and keep doing that until we reach a case that can be solved directly.

The two key components of recursion

  • Base case: the stopping point of the recursion; a condition where the function no longer calls itself.
  • Recursive case: the part of the function where it calls itself with a smaller or simpler input.

Example: Countdown

def countdown(n):
    if n == 0:          # base case
        print("Liftoff!")
    else:               # recursive case
        print(n)
        countdown(n - 1)

countdown(3)

Output:

3
2
1
Liftoff!

Example: Factorial

The factorial of a number n (written n!) is the product of all positive integers from n down to 1. For example, 5! = 5 x 4 x 3 x 2 x 1. 

def factorial(n):
    if n == 0:          # base case
        return 1
    else:               # recursive case
        return n * factorial(n - 1)

print(factorial(5))

Output:

120

Why use recursion?

Recursion allows us to write elegant solutions for problems that involve repeated patterns or structures. It is especially useful in algorithms involving trees, searching, sorting, and dividing problems into smaller subproblems.
